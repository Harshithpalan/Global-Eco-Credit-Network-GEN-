# Global Eco-Credit Network (GEN) 🌿

**Empowering Sustainable Actions through AI-Powered Verification.**

Global Eco-Credit Network (GEN) is a gamified sustainability platform designed to incentivize and reward eco-friendly behaviors. By leveraging advanced AI verification and blockchain-inspired credit systems, GEN turns environmental responsibility into an engaging and community-driven experience.

---

## 🚀 Key Features

- **🤖 AI-Powered Verification**: Uses Google Gemini AI to analyze and verify user-uploaded images of eco-friendly actions (e.g., recycling, using reusable bags, planting trees).
- **🪙 Eco-Credit System**: Earn digital credits for every verified action. Track your balance and historical impact through a real-time dashboard.
- **🎮 Gamified Achievements**: Complete challenges and reach milestones to unlock exclusive badges and level up your "Eco-Status."
- **🌍 Community Hub**: Share your sustainable journey, view a global feed of actions, and connect with other earth-conscious individuals.
- **✨ Immersive Experience**: A story-driven onboarding sequence and high-fidelity UI/UX with smooth animations and glassmorphism.

---

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/) (Vite), [Tailwind CSS](https://tailwindcss.com/) (V4), [Framer Motion](https://www.framer.com/motion/) (Animations).
- **Icons**: [Lucide React](https://lucide.dev/).
- **Database/Storage**: [Firebase](https://firebase.google.com/) (Firestore & Cloud Storage).
- **Artificial Intelligence**: [Google Generative AI](https://ai.google.dev/) (Gemini).
- **Routing**: [React Router](https://reactrouter.com/).

## ⚙️ Installation & Setup

1. **Clone the project**
   ```bash
   git clone https://github.com/Harshithpalan/Carbon-Footprint-Dashboard.git
   cd "Global Eco-Credit Network (GEN)"
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_firebase_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_id
   VITE_FIREBASE_APP_ID=your_app_id
   VITE_GEMINI_API_KEY=your_google_gemini_api_key
   ```

4. **Run Locally**
   ```bash
   npm run dev
   ```
